OPTION 1: CLOUDFLARE PAGES (RECOMMENDED - 100% FREE)
═══════════════════════════════════════════════════════════════

1. Go to: https://dash.cloudflare.com/sign-up
2. Create free account (if needed)
3. Click: Workers & Pages
4. Click: Create application → Pages → Connect to Git
5. Authorize GitHub, select repo: PersonalSite
6. Configure:
   - Build command: npm run build
   - Build output: dist/public
7. Click: Save and Deploy
8. Wait 2 minutes

✨ DONE! Your site is live at: https://your-project.pages.dev

Cost: $0/month forever (free tier)
Performance: Best (300+ edge locations)

OPTION 2: VERCEL (EXCELLENT FOR REACT)
═══════════════════════════════════════════════════════════════

Run these commands in your terminal:

npm install -g vercel
vercel login
vercel

Follow prompts (accept defaults), then:

vercel --prod

✨ DONE! Your site is live at the URL shown
